# my-ip-cn

一个简单的获取公网 IP 地址的 Node.js 库。

## 安装

```bash
npm install my-ip-cn
```

## 特点

1. 高效维护，每天检查 3 次
2. 多个 IP Provider 负载均衡以及重试机制，保证稳定性
